# expected/

This directory stores reference visual outputs for the minimal Markdown example.

`contact_sheet.png` is generated by:

```bash
python examples/minimal_markdown/run_example.py
cp examples/minimal_markdown/contact_sheet.png examples/minimal_markdown/expected/contact_sheet.png
```

The preview renderer in this example is intentionally cross-platform and
approximate. It is useful for CI/readme demonstration. For real thesis delivery,
export slides with PowerPoint (`export_pptx_png.ps1`) and then run
`make_contact_sheet.py` on the exported PNG directory.
