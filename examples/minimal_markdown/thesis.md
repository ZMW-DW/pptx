# 面向视觉模板保持的轻量级论文答辩 PPT 生成方法

## 摘要

本文提出一种面向毕业论文答辩场景的本地 PPTX 生成工作流。该方法不追求重新
设计一套全新幻灯片风格，而是优先保持用户给定 PowerPoint 模板的封面、配色、
导航条、卡片样式与字体比例，再将论文中的研究问题、方法设计、实验结果和总结
转写为适合答辩讲述的简洁页面内容。

## 研究背景

当前许多 AI PPT 工具更擅长从文档重新生成一套视觉风格。但高校答辩、实验室
汇报和单位报告往往已经有严格模板，随意改变颜色、导航、页眉或封面都会降低可
交付性。因此，更稳妥的做法是让 agent 先理解模板，再在模板原生页面上替换内容。

## 方法设计

工作流由四步组成：

1. 从论文 PDF / LaTeX / Markdown 中提取背景、问题、方法、实验和结论；
2. 分析 PPTX 模板中的版式、颜色、字号、导航与卡片结构；
3. 使用 python-pptx 或 PowerPoint COM 生成可编辑 PPTX；
4. 导出图片、生成总览图，并检查文字溢出和旧模板词残留。

## 实验结果

在最小示例中，系统生成 4 页可编辑 PPTX：封面、研究背景、方法流程和结果总结。
通过 `dump_pptx_content.py` 可以看到每页的真实文本和 shape 清单；通过
`scan_pptx_text.py` 可以确认旧模板词和临时填充内容没有残留。

## 结论

该示例说明：即使没有完整 PowerPoint COM 环境，也能在本地跑通 PPTX 生成、
结构 dump、旧词扫描和总览图生成；在正式交付时，再用 Windows + PowerPoint
补齐真实导出与文字溢出检查。
